🔑 Palabras Clave de la Lección

Monitoreo

Seguimiento constante de información específica para detectar cambios o tendencias.

Filtrado

Eliminación de datos irrelevantes o de baja calidad para quedarse con lo esencial.

Búsqueda Semántica

Búsquedas que entienden el contexto y la intención, no solo palabras sueltas.

📈 Métodos de Recopilación

No toda la información se encuentra igual. Existen diferentes técnicas:

  • Observación Digital: Monitorear redes o foros especializados.
  • Encuestas en Línea: Uso de herramientas como Google Forms o Typeform.
  • Web Scraping: Extracción automatizada de datos (para niveles avanzados).
  • Búsqueda Documental: Revisión de archivos PDF y bases de datos.

🔍 Diversidad de Motores

Dependiendo de tu objetivo, el motor de búsqueda debe cambiar:

General

Google / Bing

Para información rápida, noticias y servicios generales.

Académico

Scholar / JSTOR

Para tesis, estudios científicos y fuentes verificadas.

Privacidad

DuckDuckGo

Para búsquedas sin rastreo ni burbujas de filtro personalizadas.
